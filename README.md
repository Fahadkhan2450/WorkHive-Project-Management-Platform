# WorkHive-Project-Management-Platform
Project Description  Project Management Platform is a modern, full-featured web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The platform is designed to help teams and organizations efficiently manage tasks, track progress, collaborate with team members, and streamline project workflows.

## Features

- **Task Management**
  - Create, update, delete tasks
  - Set priority and due dates
  - Add detailed descriptions
- **User Assignment**
  - Assign tasks to team members
  - Track individual workloads
- **Todo Checklist**
  - Break tasks into smaller actionable items
  - Mark checklist items as completed
- **Attachments**
  - Upload and manage task-related files
- **Task Status Tracking**
  - Categorize tasks as Pending, In Progress, Completed
  - Track task progress visually
- **Role-based Access**
  - Admins: Full access to manage users and tasks
  - Users: Access to assigned tasks
- **Reports & Export**
  - Export tasks and reports in Excel format
- **Responsive Design**
  - Fully responsive on desktop, tablet, and mobile

## Tech Stack

- **Frontend:** React.js, React Router, Tailwind CSS, React Datepicker
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Notifications:** React Hot Toast
- **File Uploads:** Multer (or custom attachment handling)
- **Icons:** React Icons
